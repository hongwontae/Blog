2. 객체를 사용하는 두 방법
    2-1. 점 표기법
    => 정의한 키 이름을 사용하여 접근하는 정적 표현방식입니다.

    2-2. 대괄호 표기법
    => 변수나 표현식을 대괄호에 넣어서 동적으로 표현할 수 있습니다.


3. props로 함수 전달
    3-1. 전달할 함수가 인자가 없다면 그냥 참조만 합니다.
    => <Hello func1={func}/>
    
    3-2. 전달할 함수가 인자가 있다면 익명함수로 감싸야 합니다.
    => <Hello func1={()=>funcio1('Hello')}/>
