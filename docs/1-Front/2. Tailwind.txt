1. Tailwind
1-1. :root는 html 태그를 의미합니다.
1-2. html 태그에 배경색은 뷰포트 전체에 칠해집니다.
1-3. 만약 html 태그에 배경색이 없고 body 태그에 배경색이 존재한다면 
 body backgroundcolor가 뷰포트 전체에 칠해집니다.
=> html, body 둘 다 있다면 html이 전체적으로 칠해지나 컨텐츠 영역 + padding 영역은 body 색으로 칠해집니다.

1-4. @apply를 통해서 tailwind에 정의된 스타일을 조합하여 하나의 변수처럼 사용할 수 있습니다.


2. Tailwind 반응형 디자인 Prefix (브레이크 포인트)
2-1. sm :
=> 640px, 작은 화면 (작은 노트북, 태블릿)

2-2. md : 768 
=> 중간 화면 (일반 노트북)

2-3. lg : 1024
=> 큰 화면 (데스크톱)

2-4. xl : 1280
=> 더 큰 화면

2-5. 2xl : 1536
=> 매우 큰 화면

=> mobile first 개념
