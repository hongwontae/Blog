1. @Entity()
=> 엔티티 클래스 위에 정의하며 해당 클래스가 DB 테이블과 매핑됨을 알립니다.
=> 옵션을 통해 테이블 이름을 지정할 수 있습니다.
=> @Entity('blog')

2. typeorm 라이브러리 Entity에 적용될 Deco
    1-1. @Column()
        1. 일반 칼럼 데코입니다.
        2. 속성을 통해 타입지정할 수 있습니다. @Column({type : 'mediumtext'})
    
    1-2. @PrimaryGeneratedColumn()
        1. PK, 자동 증가값을 의미합니다.
        2. INT AUTO_INCREMENT

    1-3. 날짜
        1. 타입은 Data로 지정합니다.
        2. @CreateDateColumn(), @UpdateDateColumn(), @DeleteDateColumn()
        3. 모든 날짜 + 시간은 자동으로 저장됩니다. (만들어진 시간, 업데이트 된 시간)
        => 다만 DTO를 통해서 받아와서 저장 처리를 하면 자동 시간 매커니즘이 사라집니다.
        => 애초에 클라이언트에서 시간을 안주는게 좋습니다.


3. @Column({...}) 옵션
    3-1. type -> DB 칼럼 타입
    3-2. length -> DB 칼럼 속성 길이 제한
    3-3. nullable -> null 허용 여부
    3-4. default -> 기본값 설정
    3-5. unique -> 유니크 제약
    3-6. enum -> EMUN 타입 지정
